Privacy Policy

Kebijakan Privasi GUDANG BIMBEL

Pada dasarnya, kami sangat menjaga privasi pengguna GUDANG BIMBEL baik Anda sebagai pencari tempat bimbel maupun pemilik bimbel. Dalam halaman ini kami akan menjelaskan semua informasi terkait privasi dan bagaimana data pengguna kami simpan dan kelola di GUDANG BIMBEL.

Data User

  1. User dalam hal ini termasuk di antaranya pencari tempat bimbel dan pemilik bimbel.
  2. Adapun data user kami simpan secara aman di pusat data GUDANG BIMBEL.
  3. Dalam hal ini GDANG BIMBEL melindungi privasi user dan tidak akan memberikan data user GUDANG BIMBEL kepada pihak ketiga tanpa melalui persetujuan user tersebut.

Data Pengunjung Situs GUDANG BIMBEL

  1. Adapun untuk pengguna GUDANG BIMBEL yang tidak melakukan login kami menggunakan data cookies untuk menyimpan data seperti :

    • Login Anda.
    • Preferensi berdasarkan produk yang Anda klik dan Anda search.
    • Menyimpan proses yang telah Anda lakukan di GUDANG BIMBEL supaya Anda tidak perlu mengulang kembali.
    • Menyimpan data yang telah Anda isi di form supaya tidak hilang (misalnya data tempat bimbel).
  2. Selain cookies, kami juga mengambil data lokasi Anda saat ini pada saat Anda pertama kali mengakses GUDANG BIMBEL untuk kami bisa memberikan informasi Bimbel yang terdekat dari lokasi Anda sekarang. Browser akan mengirimkan notifikasi untuk meminta Anda mengaktifkan lokasi tersebut sebelumnya.

  3. Notifikasi browser. Adapun kami menggunakan pihak ketiga untuk mengirimkan notifikasi via browser, Anda tidak akan secara otomatis berlangganan notifikasi tersebut. Anda akan mendapatkan popup pemberitahuan dan disitu Anda bisa berlangganan notifikasi tersebut. Adapun notifikasi tersebut dikirimkan untuk :

    • Apabila ada balasan chat dari CS maupun pemilik pelayanan bimbel.
    • Apabila ada rekomendasi bimbel yang sesuai dengan Anda.
    • Apabila ada informasi baru yang relevan dengan Anda.
    • Menyimpan data yang telah Anda isi di form supaya tidak hilang (misalnya data bimbel).

    Anda bisa menonaktifkan notifikasi tersebut melalui browser Anda.

Data Lainnya

Adapun pengambilan data pengguna lainnya kami lakukan menggunakan survey, chat dan form yang tertera di website maupun aplikasi. Semua data yang berhubungan user adalah untuk penggunaan internal, dan tidak akan kita sebarkan ke pihak ketiga tanpa persetujuan user.

Perlindungan Data

  1. Adapun GUDANG BIMBEL menggunakan SSL (Secure Socket Layer) dan sistem lainnya untuk melindungi data.
  2. Adapun penyimpanan data juga kami lakukan di pusat data yang terpisah-pisah dan selalu diduplikasi untuk keamanan.

Hubungi Kami

Apabila terdapat pertanyaan terkait kebijakan privasi ini, silakan menghubungi kami melalui informasi kontak di bawah ini.
